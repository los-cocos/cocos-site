software requirements

  • python 2.6, 2.7 or 3.3+
  • pyglet 1.2 or newer
  • Linux, Mac OS/X or Windows

hardware requirements

To execute some effects, you will need a GPU with the GL_EXT_framebuffer_object extension.

To see if you have this extension, execute:

$ glxinfo | grep -i framebuffer
