cocos2d 0.6.9 Documentation Index¶

cocos2d is a framework for building 2D games, demos, and other graphical/interactive applications.

It works on Windows, OS X and Linux and it is meant to be used by applications written in the python language.

API Reference¶

Note

  • Type annotations in the API reference uses python 3 types

  • When text strings are expected (str in py3) str or unicode are accepted in py2.

  • To future proof your code please use unicode for text strings.
